# Advent of Code

## Download Input

### 1. set environment variables
Copy the cookie from the [adventofcode.com](adventofcode.com) website and set it as `AOC_COOKIE` environment variable.

```sh
export AOC_COOKIE='session=abc123...'
```

### 2. create a directory for the solution

The `aoc` script expects a folder structure like `2024/day01` for year 2024 and day 1.

```txt
years
├── 2020
│   ├── day01
│   ├── ...
│   └── day10
└── 2024
    ├── day01
    ├── ...
    └── day15
```
